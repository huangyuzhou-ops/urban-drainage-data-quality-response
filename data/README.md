# Public Data Directory

This directory contains figure/table source-data files released for public numerical reproduction. Raw municipal 5-min liquid-level records and original infrastructure coordinates are not redistributed.

- `figure_source_data/main_and_S1_S5/`: source data for main figures and Fig. S1-S5.
- `figure_source_data/S6_S8_reviewer_response/`: DQI robustness, missing-handling and DTFR source data.
- `figure_source_data/S9_S10_rainfall_low_variation/`: rainfall-conditioned quality and low-variation screening source data.
- `figure_source_data/S11_evidence_roadmap/`: evidence-roadmap source data.

The demonstration monitoring stream in the repository root is provided only to inspect data structure and script interfaces; it is not used to reproduce the manuscript numerical results.
