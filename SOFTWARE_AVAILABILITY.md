# Software Availability

The public repository associated with this manuscript is available at https://github.com/huangyuzhou-ops/urban-drainage-data-quality-response, with the versioned Zenodo archive at https://doi.org/10.5281/zenodo.20422917. The public software-facing materials include environment files, configuration files, an anonymization utility and released figure/table source data. These files support inspection of the public data structure and numerical checking of reported figures and tables.

Private preprocessing and analysis stages depend on restricted raw municipal monitoring records and a controlled-access processing workspace. The public archive is therefore intended for figure/table-level checking and public data-structure inspection, not for public end-to-end execution. Authorised data holders can use the released configuration and anonymization materials together with the restricted records under data-owner governance.
