# Data Directory

This directory is for public or reviewer-facing data only.

Do not place unrestricted raw monitoring records in this directory.

## Public-data policy

Allowed:

- anonymized sensor catalogue generated by `scripts/anonymize_monitoring_data.py`;
- anonymized demonstration time series if approved by the data owner;
- derived DQI tables with pseudonymous sensor IDs;
- controlled-degradation metrics;
- statistical-test outputs;
- figure source-data tables.

Not allowed:

- original liquid-level time series from the municipal monitoring system;
- original sensor IDs, node IDs or SCADA IDs;
- original coordinates or maps;
- private ID mapping between original and pseudonymous sensors;
- files copied directly from restricted operational systems.

## Expected subfolders

```text
data/
├── anonymized_monitoring_demo/
├── derived/
└── figure_source_data/
```

## Notes on transformed monitoring values

If public demonstration time series are released, liquid-level values should be interpreted as anonymized public units, not as the original metre-scale operational values. The transformation is intended to preserve temporal patterns for reproducibility demonstrations while preventing reconstruction of sensitive infrastructure states.
